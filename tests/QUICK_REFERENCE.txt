COMMON ISSUES

   Problem: "cannot find symbol"
   Fix: Add the missing method to your class
   
   Problem: "java is not recognized"
   Fix: Install JDK or add Java to system PATH
   
   Problem: "Could not find junit jar"
   Fix: Make sure junit-4.12.jar is in tests\junit\
   
   Problem: Tests won't run
   Fix: Make sure you're in the 'src' folder
